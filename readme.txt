Send a personalised message to users who have been granted a specified roles or roles

== Description ==

This plugin is perfect for member driven organisations that need to notify their members when their membership has been approved or actioned

== Installation ==

1. Upload the entire `lh-personal-welcome` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Set the required role and welcome message by navigating to Settings->Personal Welcome
